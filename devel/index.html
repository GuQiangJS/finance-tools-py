回测工具¶

简单的回测工具。根据指定的买入、卖出日期，计算收益、持仓等。

class finance_tools_py.backtest.BackTest(data, init_cash=10000, trade_amount=100, tax_coeff=0.001, commission_coeff=0.001, min_commission=5, col_name='close')¶

简单的回测系统。根据传入的购买日期和卖出日期，计算收益。

Example

>>> from datetime import date
>>> import pandas as pd
>>> from finance_tools_py.backtest import BackTest
>>> data=pd.DataFrame({
>>>     'date':[date(1998,1,1),date(1999,1,1),date(2000,1,1),date(2001,1,1),date(2002,1,1)],
>>>     'close':[4.5,7.9,6.7,13.4,15.3]
>>> })
>>> bt=BackTest(data)
>>> buysignal=[date(1999,1,1),date(2001,1,1)]
>>> sellsignal=[date(2000,1,1),date(2002,1,1)]
>>> bt.calc_trade_history(buysignal,sellsignal)
>>> bt.report()

__init__(data, init_cash=10000, trade_amount=100, tax_coeff=0.001, commission_coeff=0.001, min_commission=5, col_name='close')¶

初始化

参数

  • data (pandas.DataFrame) – 完整的日线数据。数据中需要包含 date 列，用来标记日期。 数据中至少需要包含 date 列和 close 列，其中 close 列可以由参数 colname 参数指定。

  • init_cash (float) – 初始资金。

  • trade_amount (int) – 每次交易数量。默认100。

  • tax_coeff (float) – 印花税费率。默认0.001。

  • commission_coeff (float) – 手续费率。默认0.001。

  • min_commission (float) – 最小印花税费。默认5。

  • col_name (str) – 计算用的列名。默认为 close 。 这个列名必须包含在参数 data 中。是用来进行回测计算的列，用来标记回测时使用的价格数据。

property available_cash¶

获取当前可用资金

calc_trade_history(signal_buy_date, signal_sell_date, verbose=0, allin=False)¶

计算交易记录

参数

  • signal_buy_date ([datetime.datetime]) – 计划买入日期的集合。

  • signal_sell_date ([datetime.datetime]) – 计划卖出日期的集合。

  • verbose (int) – 是否显示计算过程。0（不显示），1（显示部分），2（显示全部）。默认为0。

  • allin (bool) – 是否每次都全下（所有资金买入或所有持仓卖出）。默认为 False。 如果为 True ，则不使用构造函数中传入的 trade_amount 属性。

property history_df¶

获取成交历史的 pandas.DataFrame 格式。
