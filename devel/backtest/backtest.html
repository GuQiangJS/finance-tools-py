回测工具¶

简单的回测工具。根据指定的买入、卖出日期，计算收益、持仓等。

class finance_tools_py.backtest.BackTest(data, init_cash=10000, tax_coeff=0.001, commission_coeff=0.001, min_commission=5, col_name='close', callbacks=[<finance_tools_py.backtest.CallBack object>])¶

基类：object

简单的回测系统。根据传入的购买日期和卖出日期，计算收益。

Example

>>> from datetime import date
>>> import pandas as pd
>>> from finance_tools_py.backtest import BackTest
>>> from finance_tools_py.backtest import MinAmountChecker
>>>
>>> data = pd.DataFrame({
>>>     'code': ['000001' for x in range(4)],
>>>     'date': [date(1998, 1, 1), date(1999, 1, 1), date(2000, 1, 1), date(2001, 1, 1)],
>>>     'close': [4.5, 7.9, 6.7, 10],
>>> })
>>> bt = BackTest(data, init_cash=1000, callbacks=[MinAmountChecker(
>>>     buy_dict={'000001': [date(1998, 1, 1), date(2000, 1, 1)]},
>>>     sell_dict={'000001': [date(1999, 1, 1)]})])
>>> bt.calc_trade_history()
>>> print(bt.report())

__init__(data, init_cash=10000, tax_coeff=0.001, commission_coeff=0.001, min_commission=5, col_name='close', callbacks=[<finance_tools_py.backtest.CallBack object>])¶

初始化

参数

  • data (pandas.DataFrame) – 完整的日线数据。数据中需要包含 date 列，用来标记日期。 数据中至少需要包含 date 列、 code 列和 close 列，其中 close 列可以由参数 colname 参数指定。

  • init_cash (float) – 初始资金。

  • tax_coeff (float) – 印花税费率。默认0.001。

  • commission_coeff (float) – 手续费率。默认0.001。

  • min_commission (float) – 最小印花税费。默认5。

  • col_name (str) – 计算用的列名。默认为 close 。 这个列名必须包含在参数 data 中。是用来进行回测计算的列，用来标记回测时使用的价格数据。

  • callbacks ([finance_tools_py.backtest.CallBack]) – 回调函数集合。
