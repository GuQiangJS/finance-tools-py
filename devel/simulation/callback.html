模拟工具使用的回调¶

回调基类¶

所有回调均派生自此类型

class finance_tools_py.simulation.CallBack¶

读取数据时的回调。次类型为基类，所有回调需要派生自此基类

abstract on_preparing_data(symbol, data, context: {})¶

数据读取完成后的准备事件

参数

context (dict) – 不同的回调之间传递数据用的上下文字典。

布林带回调¶

调用 talib.BBANDS 方法生成布林线。并工具布林线计算简单的买入卖出点。

class finance_tools_py.simulation.Bolling(timeperiod, nbdevup, nbdevdn, difupstd, diflowstd)¶

调用 talib.BBANDS 方法生成布林线。并根据布林线计算简单的买入卖出点

__init__(timeperiod, nbdevup, nbdevdn, difupstd, diflowstd)¶

构造。

参数

  • timeperiod – 参考 talib.BBANDS 方法中的 timeperiod 参数。

  • nbdevup – 参考 talib.BBANDS 方法中的 nbdevup 参数。

  • nbdevdn – 参考 talib.BBANDS 方法中的 nbdevdn 参数。

on_preparing_data(symbol, data, context: {})¶
根据 timeperiod , nbdevup , nbdevdn 调用 talib.BBANDS

生成 bool_x_up , bool_x_mean , bool_x_low 列；

再分别除以 close 列，生成 diff_up , diff_mean , diff_low；

最后分别生成简易的买入/卖出比较价格，存放在 context 中，key值分别对应为 m1 和 m2

简单移动平均回调¶

根据 pandas 中的简单移动平均函数，计算平均值和标准差。

class finance_tools_py.simulation.Pandas_Rolling(min_periods=30, window=250)¶

根据 pandas 中的简单移动平均函数，计算平均值和标准差

__init__(min_periods=30, window=250)¶

构造函数

参数

  • window – 参考 pandas.DataFrame.rolling 方法中的 window 参数。

  • min_periods – 参考 pandas.DataFrame.rolling 方法中的 min_periods 参数。

on_preparing_data(symbol, data, context: {})¶

根据 min_periods 和 window 分别生成 rolling_mean 列和 rolling_std 列。

线性角度回调¶

调用 talib.LINEARREG_ANGLE 线性角度。

class finance_tools_py.simulation.Linear_Angle(max_linear_angle='MED', linear_angle=[3, 5, 10, 30])¶

调用 talib.LINEARREG_ANGLE 线性角度

__init__(max_linear_angle='MED', linear_angle=[3, 5, 10, 30])¶
参数

  • max_linear_angle (str) – 计算角度最大值取值时的参数。 可以取平均值（MEAN）为最大值，也可以取中位数（MED）为最大值

  • linear_angle ([int]) – 计算角度的日期区间。默认会分别计算 3,5,10,30 日的线性角度。

on_preparing_data(symbol, data, context: {})¶
循环对 linear_angle 指定的日期区间，调用 talib.LINEARREG_ANGLE ，生成角度值。

所有计算得到的值取绝对值，对没有值的内容填充 99 （以示角度超大）。 取到的所有值会放在 context 中，key值为 angle ； 根据 max_linear_angle 设置的值对每隔日期区间的角度制取平均值或中位数。 取得的值会放在 context 中，key值为 max_angles ；

合并多个回调生成买入卖出标记的回调¶

自动合并 finance_tools_py.simulation.Bolling , finance_tools_py.simulation.Pandas_Rolling , finance_tools_py.simulation.Linear_Angle 回调 生成的数据，并产生买入卖出信号列。

class finance_tools_py.simulation.CalcTradePoint¶

计算交易时间点

on_preparing_data(symbol, data, context: {})¶

计算买入点和卖出点。分别标记在 opt 列中，卖出为 1 ，买入为 2 。 会将买入条件和卖出条件放入context中，key值分别为 buy_query 和 sell_query 。
